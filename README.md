# Rambler-FictitiousFund
Virtual fund demo based on rambler kernel (blueprint) :  💴 Make every developer an investment manager

> **Rambler-Keep-IpAgentPool First demo** 🦅
> 让您轻松构建 基于大数据的虚拟基金决策平台, 在中国投资市场里驰骋, 黄金, A股 , 债卷 , 国债等 . Bluerint 基于 Java 版 SDK 开发. 
>
> 至于 爬虫, 网络IO, 数据挖掘, 数据存储, 实时计算 等麻烦事 则交给 **Rambler-Keep-IpAgentPool** 而我们只用做好对投资策略有益的事情.
